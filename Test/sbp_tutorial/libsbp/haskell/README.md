# [SBP Bindings for Haskell][1]

Haskell client for Swift Binary Protocol (SBP).

## Setup

To build:

    $ stack build

To install:

    $ stack install

To test and benchmark:

    $ stack test && stack bench

To deploy to Hackage:

    $ stack upload
