sbp_tutorial
==============

This is a tutorial on how to receive SwiftNav Binary Protocol (SBP)
messages from Piksi, using an STM32F4DISCOVERY Board as an example.

For more information, see the
[Piksi integration tutorial](http://docs.swift-nav.com/wiki/Piksi_Integration_Tutorial).

Submodules
==========

[libsbp](https://github.com/swift-nav/libsbp/) is a required submodule
of this git repository. You must check these out in order to build the
firmware.

Check it out using:

```shell
git submodule init
git submodule update
```
