It is a X configuration Quadcopter developed by Li Shunxing. Based on APM 2.5 board, it can serve as the master or slave. When it is regarded as a slave, it can send four motors speed to the master via UartC, and the baud rate is 115200.

Parts:
1. Remote Control: WFT06X-A
2. ESC: Hobbywing, Sky Walker, QUATTRO 25A X 4
3. Motors: SunnySky, X2212-13 KV: 980 II
4. Body: 
5. Baterry: CUAV 2200mAh, 3S, XT60
